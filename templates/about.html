About This Project

An AI-powered adaptive quiz generation system for personalized learning

Project Overview

This Adaptive AI Quiz Generator takes lecture notes (PDF/Text) as input, extracts key concepts, constructs a Concept Knowledge Graph, and generates quizzes in multiple formats (MCQ, True/False, Short Answer).

Unlike standard quiz generation tools, our system personalizes question selection using a reinforcement learning strategy based on Multi-Armed Bandits (MAB), balancing exploration and exploitation to target weak concepts while maintaining topic coverage.

Key Innovation

Concept Knowledge Graph

Concepts extracted from lecture notes are structured into a knowledge graph using:

  • TF-IDF & RAKE: Statistical keyword extraction
  • KeyBERT: Semantic keyphrase extraction
  • spaCy NER: Named entity recognition
  • SBERT: Semantic similarity for edge creation
Multi-Armed Bandit Personalization

Adaptive question selection using bandit algorithms:

  • Îµ-Greedy: Balance random exploration with exploitation
  • UCB1: Upper Confidence Bound for optimism
  • Thompson Sampling: Bayesian approach (default)

Targets weak concepts while maintaining coverage
